E-Flats

September 2015 - January 2016
A Web application that connects students and landlords

About E-Flats

Client - server application developed to connect student's and landlord's needs. The project consisted of Web application for customers as well as Desktop application for system administrators and company managers.
E-Flats is an university project, which was done in in collaboration wtih 4 other students. The development was done using Agile, SCRUM and XP, and was carefully supervised by our teacher, which has a role of a customer.
